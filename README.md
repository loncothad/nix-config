# nix-config

Features:

* NixOS configurations and modules;
* [home-manager](https://github.com/nix-community/home-manager) configurations and modules;
* [disko](https://github.com/nix-community/disko) configurations;
* [microvm](https://github.com/microvm-nix/microvm.nix) configurations.
